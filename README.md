# AWS-Cloud-Practitioner

This repository contains my learning process while preparing for the AWS Certified Cloud Practitioner (CLF-C02) exam.
It includes ndotes of my AWS Cloud Practitioner course -> summaries of AWS services, hands-on labs, and graph descriptions.

- Understand core AWS services and cloud concepts

- Gain hands-on experience using the AWS Management Console, Amazon EC2
- Prepare for the AWS Cloud Practitioner certification exam
